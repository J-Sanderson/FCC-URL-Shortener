URL Shortener Microservice

How It Works

To get a shortened URL, pass a URL to the /new/ endpoint. If your request contains a valid URL, the service will return a shortened form containing an ID number.

Examples:

  • https://miniurl-jsanderson.glitch.me/new/http://www.google.com
  • https://miniurl-jsanderson.glitch.me/new/http://www.bbc.co.uk
  • https://miniurl-jsanderson.glitch.me/new/not a valid URL

To return an existing shortened URL, pass the ID number to the /s/ endpoint. If your request contains a saved URL, the service will redirect you to it.

Examples:

  • https://miniurl-jsanderson.glitch.me/s/1
  • https://miniurl-jsanderson.glitch.me/s/5
  • https://miniurl-jsanderson.glitch.me/s/0
